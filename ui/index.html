Visualisation des données des champions de League of Legends

Le but de cette application est d'avoir une représentation des données des champions du jeu League Of Legends™, dans le but de comparer la fréquence d'apparition de certaines de leurs caractéristiques mais aussi de les classer par ordre de pourcentage de victoire ou encore d manière générale du meilleur au moins bon (bien que cela soit subjectif).

Quelques précisions importantes pour comprendre les données !

League Of Legends est un jeu complexe pour qui n'y est pas initié et il peut parfois être compliqué d'en comprendre tous les termes, voici une liste des termes auquels vous serez confrontés avec une explication afin que vous puissiez être à même d'en comprendre le contenu.

  • Champions : les personnages que le joueur incarne.
  • Rôle : le poste que le champion occupe (ils sont au nombre de 5, sans entrer dans les détails)
  • WinRate : le pourcentage de victoire du champion.
  • Classe : la catégorie dans laquelle se situe le champion (combatant, mage, assassin tank ou encore tireur).


Pour plus d'informations vous pouvez toujours lire les bases, sur le site officiel du jeu (attention, ce n'est absolument pas une invitation à y jouer).
